# PondPedia

PondPedia is an Android application that aim to help improve the fish farming practice or aquaculture in Indonesia. 
Users can add their pond data and update it to regularly maintain it in a better way

## Features.
- Add Pond to the Pond Management Database.
- Browse your the created ponds in a grid layout.
- View the details of pond managed
- Dark and Light mode theme options.
- Homepage customization.

## Want to try it in Android Studio?
1. Clone the repository
```bash
git clone https://github.com/pondpedia/mobile-development.git
```
2. Open the project in Android Studio
3. Build the project.
4. Run the application on an Android device or emulator.
